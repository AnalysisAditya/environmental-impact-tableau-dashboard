# Environmental Impact Tableau Dashboard 🌍📊

An interactive Tableau dashboard analyzing environmental impact data across industries and regions — covering carbon emissions, water consumption, energy usage, waste generation, recycling rates, air quality, and regulatory compliance.

## 📁 Files in this Repo
- `TABLEAU_DASHBOARD_presentation.twbx` – Packaged Tableau workbook (data + visuals included)
- `Environmental_Impact_BRD__1_.docx` – Business Requirements Document outlining project scope, goals, and specs
- `dashboard-preview.png` – Screenshot of the dashboard (optional)

## 🎯 Project Overview
This project visualizes the environmental footprint of operations across multiple industries and regions, helping sustainability teams, compliance officers, and executives make data-driven decisions to reduce negative impact.

## 📊 Key Metrics Tracked
- Carbon Emissions (kg)
- Water Consumption (liters)
- Energy Usage (kWh)
- Waste Generated (kg)
- Recycling Rate (%)
- Air Quality Index (AQI)
- Compliance Level

## 🔍 Dashboard Features
- Bar chart: Carbon emissions by region
- Line chart: Energy usage trends over time
- Heatmap: Recycling rates by industry and region
- Pie chart: Compliance level distribution
- KPI indicators for key regional metrics
- Filters: Region, Industry, Product Type, Compliance Level
- Interactive drill-downs and hover tooltips

## 🛠 Tools Used
- Tableau Desktop (2025.2)
- Microsoft Excel (source data, 50,000 records)

## 🚀 How to Use
1. Download `TABLEAU_DASHBOARD_presentation.twbx`
2. Open with [Tableau Desktop](https://www.tableau.com/products/desktop) or [Tableau Reader](https://www.tableau.com/products/reader) (free)
3. Explore filters and interact with the visuals
4. Refer to the BRD document for full project scope and requirements
